SQL Avanzado

  • Valor: $45.000 ARS.
  • Tiempo de dedicación necesario: 60 horas.
Descripción del curso:

Este curso ofrece una formación avanzada en SQL, enfocándose en técnicas y herramientas para trabajar con bases de datos complejas.

Aprenderás a optimizar consultas, manejar subconsultas, operaciones avanzadas de JOIN, CTEs, índices, vistas y procedimientos almacenados.

Ideal para desarrolladores, analistas de datos o administradores de bases de datos que busquen mejorar su dominio de SQL.

Requisitos previos:
  • Conocimientos básicos de SQL y manejo de bases de datos.
  • Experiencia en escritura de consultas SELECT, INSERT, UPDATE y DELETE.
  • Familiaridad con conceptos de normalización y diseño de bases de datos.
  • Acceso a un gestor de base de datos como MySQL, PostgreSQL, SQL Server u Oracle.

Contenidos por clase

Introducción
  1. Presentación de SQL avanzado: importancia en bases de datos, optimización y análisis de datos complejos.
  2. Instalación y configuración de un entorno de base de datos (MySQL, PostgreSQL, SQL Server, Oracle).
  3. Herramientas para desarrollo SQL: uso de clientes SQL, IDEs especializados y herramientas gráficas.
Unidad N°1
  1. Subconsultas y consultas anidadas: definición, tipos (scalar, correlated) y casos de uso.
  2. Uso avanzado de funciones agregadas y de ventana (OVER(), PARTITION BY, RANK(), ROW_NUMBER()).
  3. Optimización de consultas: índices, EXPLAIN, análisis de planes de ejecución.
Unidad N°2
  1. Operaciones complejas de JOIN: INNER JOIN, LEFT JOIN, RIGHT JOIN, FULL OUTER JOIN, CROSS JOIN y combinaciones múltiples.
  2. Uso de CTEs (WITH) y tablas temporales para estructurar consultas complejas.
  3. Técnicas avanzadas de actualización: MERGE, UPSERT y transacciones (BEGIN, COMMIT, ROLLBACK).
Unidad N°3
  1. Gestión avanzada de índices, constraints y triggers para mejorar rendimiento y consistencia.
  2. Uso de vistas y materialized views para optimizar consultas recurrentes.
  3. Monitorización de bases de datos: análisis de rendimiento, locking, deadlocks y tuning de consultas.
