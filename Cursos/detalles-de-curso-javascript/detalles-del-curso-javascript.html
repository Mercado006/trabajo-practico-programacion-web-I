Fundamentos de JS

  • Valor: $40.000 ARS.
  • Tiempo de dedicación necesario: 50 horas.
Descripción del curso:

Este curso brinda una introducción completa al lenguaje de programación JavaScript, cubriendo los fundamentos esenciales para el desarrollo web.

Aprenderás a crear aplicaciones simples, comprender la estructura de un programa Java, y trabajar con variables, tipos de datos, control de flujo y métodos.

Ideal para principiantes que quieran iniciarse en la programación y el desarrollo web.

Requisitos previos:
  • Conocimientos básicos de navegación web y uso de un navegador.
  • Familiaridad con conceptos básicos de programación (variables, condicionales, bucles).
  • Conocimientos elementales de HTML y CSS.
  • Manejo básico de un editor de código (por ejemplo, VS Code).

Contenidos por clase

Introducción
  1. Presentación de JavaScript: historia, características y ámbitos de uso (web, backend con Node.js, aplicaciones híbridas, etc.).
  2. Instalación y configuración del entorno de desarrollo (navegador, Node.js, editor de código).
  3. Primer programa en JavaScript: estructura básica, ejecución en consola y en navegador.
Unidad N°1
  1. Variables y constantes: var, let, const y su alcance (scope).
  2. Tipos de datos primitivos y no primitivos: string, number, boolean, null, undefined, object, array.
  3. Operadores: aritméticos, lógicos, relacionales, ternario y operadores especiales (typeof, instanceof).
Unidad N°2
  1. Estructuras condicionales: if, else, switch.
  2. Bucles: for, while, do-while, for...in, for...of.
  3. Funciones: declaración, expresión, arrow functions, parámetros y valores por defecto.
Unidad N°3
  1. Objetos: creación, acceso a propiedades, métodos y destructuring.
  2. Arrays: métodos comunes (map, filter, reduce, find, forEach), spread operator y destructuring.
  3. Manejo de errores: try, catch, finally, y buenas prácticas de depuración.

Docente

Brendan Eich

Conocido como el creador de JavaScript, Brendan Eich es un referente mundial en desarrollo web y estándares de programación. Con más de 30 años de experiencia, ha trabajado en Mozilla y ha influido profundamente en la evolución del lenguaje.

Su claridad para explicar conceptos y su pasión por la enseñanza lo convierten en un docente excepcional para este curso de JavaScript.
